# Usefull command : 

## to install all the dependencies

```bash
make install
```

(you maybe need to install yarn)

## to compile your asset 

```bash
make encore
```

and if you want Webpack encore to listen to your change

```bash
make encore-watch
```

## to start a server

```bash
make server
```

## to clear symfony cache

```bash
make cc
```
